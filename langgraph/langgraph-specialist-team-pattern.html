Chapter 07 · Pattern 2

Specialist teams make agents feel like departments.

Use this pattern when related agents belong together. A refund case may need an order team, product quality team, and retention team, each with its own internal workflow and clean output.

3 Code shape

Specialist teams are often subgraphs.

order_team = StateGraph(RefundState)
order_team.add_node("lookup_order", lookup_order)
order_team.add_node("check_window", check_return_window)

quality_team = StateGraph(RefundState)
quality_team.add_node("inspect_claim", inspect_claim)
quality_team.add_node("recommend_action", recommend_action)

parent.add_node("order_team", order_team.compile())
parent.add_node("quality_team", quality_team.compile())
Design rule

Give every team a clear interface: what state fields it expects, what fields it writes, and what summary it returns to the parent graph.
